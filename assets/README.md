# Rusty Pinch Media Bundle

This folder contains the branding media for the clean Rust package.

- `banner-rusty-pinch.png`: hero/banner image
- `logo-rusty-pinch.png`: primary logo
- `media-rusty-pinch.png`: social/media image
- `favicon_rusty-pinch/`: favicon package (`png`, `ico`, `webmanifest`)

Use these assets directly in docs, release notes, landing pages, and packaged distributions.
